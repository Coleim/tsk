## Color Reference

The statistics view uses the existing Catppuccin-inspired theme colors:

| Element | Color | Usage |
|---------|-------|-------|
| To Do bar | Blue | Status distribution |
| In Progress bar | Yellow/Peach | Status distribution |
| Done bar | Green | Status distribution |
| High priority | Red | Priority breakdown |
| Medium priority | Yellow | Priority breakdown |
| Low priority | Blue | Priority breakdown |
| None priority | Gray/Overlay | Priority breakdown |
| Overdue | Red (warning) | Due date metrics |
| Completion bar | Green gradient | Progress indicator |
| Empty/background | Dark overlay | Unfilled portions |

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `S` | Toggle statistics overlay |
| `Esc` | Close overlay |
| `S` (when open) | Close overlay |
