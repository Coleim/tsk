# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-08

### Added
- Initial release
- Three-pane Kanban workflow (To Do, In Progress, Done)
- Vim-style keyboard navigation (j/k/h/l)
- Arrow key navigation support
- Task creation and editing
- Task deletion with undo
- Task movement between panes
- Priority levels (High, Medium, Low, None)
- Labels for task categorization
- Due date support
- Multiple board support
- Board creation, switching, renaming, and deletion
- Search across title, description, and labels
- Filter by priority and labels
- Sort by priority
- Archive for completed tasks
- Export/Import boards as JSON
- Full undo/redo support (20 action history)
- Two-line status bar with context-sensitive shortcuts
- Help overlay (? key)
- Auto-save with dirty tracking
- Backup before destructive operations
- Responsive layout with terminal resize handling
- First-run welcome screen
